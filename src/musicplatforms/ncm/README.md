# JSNCM
一个平台无关、运行时无关的针对[pyncm](github.com/mos9527/pyncm)项目的`JavaScript`重实现，所有代码使用ChatGPT o3-mini-high从Python进行近乎1:1的转换，并进行人工微修，可以快速移植到任何支持运行JS代码的设备上或使用JS的项目中。

## 如何使用？
1. 下载
2. 放到你的项目中
3. 根据你的平台实现platform.js中的接口
4. Enjoy It
